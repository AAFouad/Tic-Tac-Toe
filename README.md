# Tic-Tac-Toe

A simple Java implementation of the classic Tic-Tac-Toe game for two players.

## Features

- Two-player gameplay
- Simple command-line interface
- Win detection and draw handling

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AAFouad/Tic-Tac-Toe.git
   cd Tic-Tac-Toe
   ```

2. Compile and run:
   ```bash
   javac *.java
   java TicTacToe
   ```

## How to Play

- Players take turns marking spaces with X or O
- First player to get three marks in a row (horizontally, vertically, or diagonally) wins
- If all spaces are filled with no winner, the game is a draw
