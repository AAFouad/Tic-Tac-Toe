# Tic-Tac-Toe

A simple yet engaging implementation of the classic Tic-Tac-Toe game in Java. Play against the computer or a friend in this timeless strategy game.

## Features

- **Interactive Gameplay**: Play against the computer AI or another player
- **Simple Interface**: Intuitive command-line or GUI-based interaction
- **Java Implementation**: Clean, object-oriented code written in Java

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Git (for cloning the repository)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AAFouad/Tic-Tac-Toe.git
   cd Tic-Tac-Toe
   ```

2. Compile the Java files:
   ```bash
   javac *.java
   ```

3. Run the game:
   ```bash
   java TicTacToe
   ```

## How to Play

1. The game is played on a 3×3 grid
2. Players take turns marking spaces with X or O
3. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins
4. If all spaces are filled and no player has won, the game is a draw

## Game Rules

- **Win Condition**: Three marks in a row (horizontal, vertical, or diagonal)
- **Draw Condition**: All nine spaces are filled with no winner
- **Valid Moves**: Players can only mark empty spaces
- **Turn Order**: Players alternate turns
