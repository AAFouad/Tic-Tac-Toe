# Tic-Tac-Toe

A simple Java implementation of the classic Tic-Tac-Toe game for two players.

## Features

- Two-player gameplay
- Simple command-line interface
- Win detection and draw handling

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AAFouad/Tic-Tac-Toe.git
   cd Tic-Tac-Toe
   ```

2. Compile and run:
   ```bash
   javac *.java
   java TicTacToe
   ```
